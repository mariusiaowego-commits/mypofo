- **布局**：单列竖排，每张卡片占满宽
- **字号**：标题 56pt / 卡片 32pt / 正文 19pt（手机友好）
- **典故串**：dizical（笛卡尔）/ sonquiz（g/q 镜像）/ mofang（给女儿） / moni（monitor）/ mole-vibe（摸了）/ chronocare（给父母）/ creditsbar（5 个 provider）
