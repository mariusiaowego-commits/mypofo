# 截图占位

把 9 张代表项目特色的截图放进这个目录。文件名必须严格匹配:

| 项目 | 文件名 | 建议截什么 |
|---|---|---|
| dizical | `dizical.png` | iPad kid-app 练习页 |
| mofang | `mofang.png` | 终端 TUI 成绩曲线 |
| chronocare | `chronocare.png` | Dashboard 长期趋势图 |
| literary-maps | `literary-maps.png` | 地图界面（湘西/马孔多等点位） |
| mole-vibe | `mole-vibe.png` | 终端 TUI 行情 |
| moni | `moni.png` | 研究简报/周报 |
| sonquiz | `sonquiz.png` | 还原后的 Word 原版式样张 |
| uiux-asset-library | `uiux-asset-library.png` | design tokens 索引页 |
| creditsbar | `creditsbar.png` | Nothing White 主题 HUD |

图片要求:
- PNG 或 JPG
- 宽高比建议 16:9 或 4:3
- 单图 < 500KB (出图时控制)
- 在米黄纸卡片里展示, 浅色背景效果最好
